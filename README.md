# registration-app

## Quickstart

### Back-End Server (:8080)

To run dev server:

```
mvn spring-boot:run
```

### Front-End Web App (:3000)

To run dev server:

```
npm start
```

To run tests:

```
npm test
```

## Dependencies
* node, npm
* webpack, webpack-dev-server
* karma
* protractor
* typings
* typescript

## Technologies Used
* Angular 2
